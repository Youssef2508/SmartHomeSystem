Smart Home System (ATmega32)

🚀 Features

- Temperature Monitoring using LM35
- Light Detection using LDR
- Flame Detection
- Automatic Fan Control
- Buzzer Alarm System
- LCD Display

🛠️ Components

- ATmega32
- LM35 Sensor
- LDR Sensor
- Flame Sensor
- DC Motor
- Buzzer
- LCD

⚙️ Drivers

- GPIO
- ADC
- PWM
- Timer

📂 Structure

- Code (Embedded C)
- Simulation (Proteus)

▶️ How it Works

- Reads sensors data
- Processes logic
- Controls outputs (Motor, Buzzer, LEDs)
